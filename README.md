# Restaurant Co-Pilot Project

This project aims to develop a full-stack restaurant management application.

## Project Structure
- **backend/**  
  - **models/**  
  - **services/**  
  - **controllers/**  
  - **routes/**  
- **frontend/**  
  - **components/**  
- **docker/**  
- **tests/**  
- **seed/**  
- **docs/**  

## Setup Guide
1. Clone the repository.
2. Navigate to the backend and frontend directories and install the dependencies.
3. Build Docker images and run the containers.

## Features
- User Authentication
- Restaurant Management
- Order Processing
